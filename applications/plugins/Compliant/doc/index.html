Introduction

This plugin introduces a constraint-based approach. Hard constraints are solved using a classical Delassus operator \( J M^{-1} J^T \), while soft constraints are handled by adding compliance to the same operator: \( J M^{-1} J^T + C \).

Examples

  • ComplianceTest.scn : a simple pendulum with one particle.
  • ComplianceCloth.scn  : a cloth-like triangle structure
.

Project structure

  • the C++ code is in the root directory
  • the documentation starts in doc/index.html and it is linked to the code documentation
  • the examples are in examples/ . The first one is MyPlugin.scn, and when runSOFA opens it, an html page pops up, with a link to the plugin documentation.
