Simple Shader (6/7)




Description:

The final tweak to do for visualizing in the use of the Shaders, in other ways replace OpenGL standard operations by your own in 3 differents parts in the rendering pipeline.
Key points

The way to apply shaders in Sofa is done with the object called OglShader. It will call its shader code on all its Visual Model (OglModel) in its node and its following child nodes. In that way, you can have several shaders in your scene. The mandatory parameters are vertFilename and fragFilename where you give the path to the vertex and fragment GLSL code.
