Mesh Loading and Rendering 1 (1/7)




Description

The objective of this first tutorial about visualization is to load an obj file and render it in the viewer.

Key points

OglModel is the main component for rendering file. This is where all geometrical informations (vertices, faces ...) are given to OpenGL, which render it afterwards. Vertex Buffer Object is used (if supported) to enhance speed. Just put an OglModel in the scene and fill the field filename with the path to your OBJ file.
Note:

In Sofa, visual objects like OglModel are shown in the "Visual" graph, instead of the normal scene Graph.
