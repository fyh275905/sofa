Mesh Loading and Rendering 2 (2/7)




Description

This tutorial follows directly the previous one, and shows how to load mesh files which are not in OBJ format.

Key points

As we saw previously, OglModel can render simply OBJ file. This was done directly in this component because OBJ is widely used. But if you have less common format files, there is a different method to render them.
MeshLoader is used to load several kind of files. Connect its loaded data in OglModel which will used it to visualize. In this example, we render a VTK file, using MeshVTKLoader.
