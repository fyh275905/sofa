Lighting (4/7)




Description

We will see now how to manage lights.

Key points

By default, a (positional) is present in the OpenGL Viewer, for those who do not want to mess with it. But if you want to tweak your scene, you may want to modify this light or add other lights.
In order to do that, you have to add first a LightManager without any kind of parameters (we will see later why we need it). After that, you can add until the maximum of lights permitted by OpenGL, which must be 8 on most common systems.
There are 3 kinds of lights in SOFA. It's basically the same than in OpenGL :
  • PositionalLight (main parameter : position) ;
  • DirectionalLight (main parameter : direction) ;
  • SpotLight (main parameters : position and direction) .
color is a common parameter for all these 3 lights. SpotLight has other non-essential parameters : cutoff and exponent (default value may suffice).

Note:

LightManager and *Light must be in the root node if you want all your scene lighted.
