Creation of a Pendulum (8/8)

Description

We do not use explicit integration anymore, but implicit integration instead.
Key points

We removed the EulerSolver and replaced it by another ODE Solver. The EulerImplicitSolver contains the algorithm used to update particle positions and velocities at each time step. It uses an auxiliary component, the CGLinearSolver, to solve equation systems.
Results

The behavior is slightly different, try to interact with the mouse with both ODE solver. Modify the parameters of the ODE Solver and the Linear Solver.
