Creation of a Pendulum (2/8)


Description:

In the previous scene, no solver was present, so no simulation was done! We add an ODE solver in the Root node of the Simulation. We provide several ODE solvers, with different integration scheme: take a look to our tutorials on Solvers, and see also all the components available in the section OdeSolver of the Modeler.
Components:

We will use the classic explicit euler solver EulerSolver to update particle positions and velocities at each time step: In Sofa, a solver acts on all the objects placed below.

In Sofa:

This time, when you animate the scene, the particule falls along the direction of the gravity.
The gravity is specified in the root node of the simulation. We use a default value of (0, 0, -9.81). Changing the gravity on child nodes of the simulation won't do anything as at each time step, the gravity, and other parameters (time, dt...) is overwritten using the root value.
If you want to specify a specific gravity for a node and all its children below, use the component Gravity. Using the Modeler, find the component in the list, select it: an example is available.
