Creation of a Scene (4/9)


Description

Now that we have a topology, a mechanical object well initialized, it is time to simulate the deformation of the object. To do so, we add a ForceField to the scene. You have to put a compatible forcefield: compatible means, it must use the same template as your MechanicalObject (here we used Vec3d: you won't be able to add force fields using Rigid types for instance), and some force fields rely on a given topology. Most of the time, their name informs you about this topology.
In this scene, the SparseGridTopology creates hexahedra corresponding to the cells of the grid: thus you can use all Hexahedron based ForceFields.
Key points

We chose to model our deformation using Finite Elements: HexahedronFEMForceField.

Now you should see the hexahedra corresponding to the Finite Elements. When you simulate, you will be able to see the deformation of the cells of the grid.
Related

Modify the young modulus of the FEM force field: the higher, the stiffer will be the model.
Modify the poisson ratio (between 0 and 0.5).
