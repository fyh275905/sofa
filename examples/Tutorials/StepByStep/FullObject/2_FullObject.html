Creation of a Scene (3/9)


Description

One of the quickest way to model a deformable body is to embed it inside a regular grid. We will only simulate the points from the cells of the grid. The description of this grid is made inside a Topology which describes how the particles are connected.
Key points

We add a SpareGridTopology: it only keeps the full cells of this embedded grid.
Now that we have a topology in the scene, the MechanicalObject will rely on it to initialise itself. To use this component, you have to:
  • specify the 3d model using the parameter fileTopology
  • specify the resolution of the grid using the parameter n

Results

Several particules are created now, representing your deformable object: it embeds the visual model we put. If you animate, they fall against the Plane we defined previously. As no internal force, other than a plane, is defined, the original shape is not kept.
Related

Change the 3d mesh for the SparseGridTopology, and the resolution of the grid.
