Topological Mapping (5/7)


Description

This step shows that it is possible to define mechanical behavior directly on child topology. this can be usefull as often the surfacique mesh is only a small part of the entire topology.
Key points
  • Springs are added to the edges on surface of the mesh using TriangularBendingSprings.
  • TriangleModel component can be added to interact with the model (use shift + left mouse button).

Results

In this step, edges on the surface of the mesh have springs, thus the cylinder doesn't bend as mush as in previous steps. The interest is that elements in the rest of the cylinder are not taken into account, leading to faster computation.
