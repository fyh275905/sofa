Explicit Euler Solver

Starting scene Normal simulation
Description:

In Sofa, EulerSolver denote the ODE solver using the explicite integration scheme for computing of the dynamic system. This integration scheme resolve the system in relation to time : given the mechanical state at time Tn, compute the state at time Tn+1=Tn + dt.

Recomanded use :

According to Taylor expansion, the error for first derivative approximation of a function f(t) is in relation to dt and f"(t). The term f"(t) can represent the dynamic (forces, acceleraion) of the system. Thus, there are two important criteria for the stabilization when using the explicit EulerSolver :

  • The time step dt.
  • The dynamic of the system.
