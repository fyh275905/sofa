Newmark Implicit Solver

Description

The NewmarkImplicitSolver can be used to solve ODE system, using the Newmark implicit integration scheme. Step size of the discratisation correspond to the timeStep of SOFA.
At each step of the scheme, the linear equation systems obtained have to be solved by an auxiliary component, the CGLinearSolver, which can be of different kind (conjugated gradient, ...).
